# Emotion Detector using Watson NLP

An AI-powered web application that detects emotions from user input text using IBM Watson NLP.

---

## 📌 Overview

This project is a **text-based emotion detection system** that analyzes user input and identifies emotions such as:

- Joy 😊  
- Sadness 😢  
- Anger 😡  
- Fear 😨  
- Disgust 🤢  

Unlike basic sentiment analysis (positive/negative), this application provides **fine-grained emotional insights**.
